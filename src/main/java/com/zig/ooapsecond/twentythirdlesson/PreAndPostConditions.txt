1. Предусловия нельзя усиливать

Есть абстрактный класс "Устройство печати" и наследники "Печатная машинка" (аналоговая) и "Электронная печатная машинка",
у котрых есть метод для печати. Если для "Печатная машинка" просто вызывается метод печати, то для  "Электронная печатная машинка"
пришлось бы усилить предусловие, что она включена в сеть.

2. Постусловия нельзя ослаблять

Для того же примера с печатными машинками, метод "Закончить работу" в наследнике "Электронная печатная машинка"
в постусловии может быть усилен проверкой выключена ли машинка от сети.
