1. Игровое поле
Функции:
- Хранение текущего состояние сетки элементов;
- Первичное заполнение поля случайными элементами;
- Предоставление методов по перестановке элементов (?).

2. Элемент/ячейка
- Ячейка поля с типом элемента (A, B, C, D, E).

3. Пользовательский интерфейс
Функции:
- Ввод/вывод состояния поля, статистики, ходов, ошибок в консоль;

4. Управление (движок)
Функции:
- Обрабатывает ходы игрока, перестановки, результаты;
- Проверяет завершение игры (нет возможных ходов);
- Управляет историей ходов.

5. Класс бонусов
Функции:
- Управление бонусами (активация, применение "вида" бонусов...).

6. Класс подсчёта очков
Функции:
- Подсчет и хранение очков;
- Статистика (очки, ходы).
