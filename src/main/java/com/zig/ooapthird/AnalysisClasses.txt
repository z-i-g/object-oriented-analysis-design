Классы анализа:
•	GameBoard - игровое поле;
•	Element - базовая ячейка поля (A, B, C, D, E);
•	Combination - ряд/столбец из трёх и более одинаковых элементов;
•	Bonus - особая возможность/награда, за сложную комбинацию;
•	Score - числовой показатель "успеха" игрока;
•	Cell -  ячейка игрового поля;
•	UserInterface - игровое поле;
•	UserStep - информация о ходе пользователя.
